## Why This Format

Write content in Markdown, then generate HTML so every post keeps the same layout and CSS.

### Supported Markdown

- Headings (`#`, `##`, `###`)
- Paragraphs
- Blockquotes (`> quote`)
- Ordered and unordered lists
- Inline code like `python -m http.server 8000`
- Fenced code blocks
